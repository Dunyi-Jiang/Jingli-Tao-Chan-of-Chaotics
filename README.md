## 道统核心
遁一·BSPP核心纲领声学落地代码，飙刹推拉，道统为核，术用为翼。

### 🔍 理论溯源
本仓库是BSPP（飙刹推拉）核心理论的开源技术实现。理论全文已发表于SSRN预印本平台，全球可查可引用：
[BSPP (Biao-Sha-Push-Pull) Core Theory: A Cross-Disciplinary Cognitive Framework Originated from Migrant Worker's Intuition](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6168547)

### 🎯 核心定位
这是一套从农民工直觉出发，横跨声学、金融、神经科学的跨学科认知框架。它以打桩机的6Hz节奏和甘蔗的分段生长为意象，揭示了复杂系统的动态平衡规律：
- **飙（Biao）**：能量释放/扩张
- **刹（Sha）**：能量收敛/抑制
- **推（Tui）**：正向协同
- **拉（La）**：反向制衡

### 🛠️ 代码功能
- 实现SAP（分段交替包）认知模型，在167ms微时间尺度上具象化BSPP理论
- 生成符合神经同步的“单轨量子态”音频信号
- 提供金融市场订单流平滑算法的同源验证
