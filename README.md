# 惊璃混沌道场 | Jingli Tao-Chan of Chaotics
道统核心：遁一BSPP核心纲领声学落地代码，飙刹推拉，道统为核，术用为量。


## 项目定位
以BSPP声学协议为根基，融合金融SAP模型、时间存证体系的数字道场――既是技术实现的代码集合，也是混沌哲思的显化载体。


## 目录结构
| 英文目录 | 道场语境 | 功能说明 |
|----------|----------|----------|
| src/ | 【心流】 | 源代码核心实现（BSPP声学协议、功能逻辑） |
| docs/ | 【典律】 | 协议规范、存证文档（含BSPP-CORE/SAP模型、时序契证） |
| papers/ | 【道论】 | 核心思想与模型论文（如甘蔗模型等混沌哲思） |
